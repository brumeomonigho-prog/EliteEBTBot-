# EliteEBTBot-Build a production-ready Telegram bot in Python.

Bot name: EliteEBTBot

Features:
1. /start command
2. Show real estate properties
3. Store users (name, phone, telegram id)
4. Admin panel via Telegram commands
5. Add/edit/delete properties
6. Prepare for crypto payment integration (USDT TRC20)
7. Clean modular structure:
   - bot.py
   - handlers/
   - database/
   - config.py
8. Use python-telegram-bot v20+
9. Environment variables for secrets
10. Ready to deploy on Koyeb

Make it simple and scalable for a real estate business.
